# 🌌 3D Solar System

交互式 3D 太阳系模型 — 基于 Three.js 构建，浏览器打开即用。

## ✨ 功能

- **太阳 + 八大行星 + 月球** — 写实比例与纹理
- **星空背景** — 多层动态星空 + 闪烁效果
- **鼠标操控** — 拖拽旋转、滚轮缩放、右键平移
- **点击天体** — 显示信息弹窗（直径、自转周期、公转周期）
- **控制面板** — 轨道开关、视角缩放、速度调节、星空闪烁开关
- **太阳辉光** — 多层 Additive 叠加光晕 + 呼吸脉动效果
- **流星效果** — 高速带拖尾的流星粒子

## 🚀 使用

1. 下载 `solar-system.html`
2. 用浏览器打开（Chrome / Edge / Firefox）
3. 无需安装任何依赖，全部资源通过 CDN 加载

## 🛠 技术栈

- **Three.js** r170 — 3D 渲染引擎
- **Canvas 2D** — 程序化纹理生成
- **纯 CSS** — 毛玻璃 UI 面板
- **ES ImportMap** — 零本地依赖
